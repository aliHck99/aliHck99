\section{Projects}

\textbf{Active Directory Penetration Testing Lab}\\
{\small Impacket, CrackMapExec, Hashcat, Enum4linux, Nmap, Kali Linux, Windows Server 2025}\\
{\small \href{https://github.com/naim-ali27/active-directory-pentest-lab}{github.com/naim-ali27/active-directory-pentest-lab}}

\begin{itemize}
  \item Built and attacked a self-hosted Windows Server 2025 AD environment with \textbf{100+ accounts}
    --- full attack chain from scanning and enumeration through credential theft, lateral movement,
    and domain compromise.
  \item Performed \textbf{AS-REP Roasting} via impacket-GetNPUsers and cracked two domain accounts
    offline using Hashcat.
  \item Ran \textbf{DCSync} via impacket-secretsdump to extract all domain password hashes,
    including Administrator, krbtgt, and service accounts, without interactive access to
    the domain controller.
  \item Chained \textbf{Pass-the-Hash} over SMB into a \textbf{SYSTEM shell} using impacket-wmiexec without
    recovering plaintext credentials.
  \item Mapped the attacks to MITRE ATT\&CK: T1046 (Network Service Discovery), T1087
    (Account Discovery), T1558.004 (AS-REP Roasting), T1003.006 (DCSync), T1550.002
    (Pass-the-Hash) and documented five AD misconfigurations with remediation notes.
\end{itemize}
